## 三、子任务详细计划

### 📌 子任务1: 接入国产模型 API

**目标**: 不替换本地 Ollama，而是**并行支持**云端国产模型。用户随时切换。

```
                ┌─ 本地 Ollama (免费/隐私)
                │    qwen2.5:7b · deepseek-r1:8b · glm4:9b
                │
Javis ──模型路由─┼─ 国产 Cloud API (更快/更强)
                │    DeepSeek · GLM · Kimi · 通义千问
                │
                └─ 海外 Cloud (可选)
                     OpenAI · Claude
```

**实施**:
1. 扩展现有设置面板, 加入国产模型列表
2. 每个模型填 API Key 即可切换
3. 支持 DeepSeek/GLM/Kimi/通义千问 — 都是 OpenAI 兼容接口, 只需改 base_url
4. 本地优先 → 本地失败自动降级云端 → 云端失败回退本地

**代码改动**: 
- `config.yaml` 加国产模型段
- `core/llm_client.py` 加降级链
- `web/js/app.js` 设置面板加选项

### 📌 子任务2: JARVIS 电影风格语音

**目标**: 替代现在的机械 TTS, 让 Javis 听起来像电影里那样 ── 英伦腔、稳重、低音。

**现在的问题**:
- edge-tts 的 "Xiaoxiao" 是新闻播报腔, 像导航不像管家
- STT 用 faster-whisper base 模型准确率一般
- 没有情感、没有语气变化

**方案**:

```
语音输入 (耳朵)                    语音输出 (嘴巴)
│                                  │
├─ faster-whisper medium           ├─ 方案A: edge-tts "Ryan" (英伦男声)
│  (升级模型提升准确率)              │   免费, 英文好, 中文一般
│                                  │
├─ 加 VAD 静音检测                 ├─ 方案B: Coqui TTS (本地)
│  (自动判断说完)                   │   完全本地, 可调音色
│                                  │
└─ 加上下文理解                     ├─ 方案C: OpenAI TTS (需Key)
   (结合对话历史纠错)                │   音质最好, 中文OK
                                   │
                                   └─ 方案D: 预录 Javis 固定台词
                                      "Yes, sir" / "Right away" / "Done"
                                      关键时刻用, 最有电影感
```

**推荐组合**: 方案A + 方案D — 日常用 edge-tts 英伦腔, 关键时刻播预录台词。

**给语音加 "智能体辅助"**:
- Javis 说话时能感知你是否在打断 (双工对话)
- 根据上下文调整语气 (严肃任务 ≠ 闲聊)
- 任务完成/失败时播不同的提示音

### 📌 子任务3: 当前缺陷清单 + 方向

| 等级 | 缺陷 | 影响 | 方向 |
|------|------|------|------|
| 🔴 | **LLM 被调两次** | 浪费50% token, 慢一倍 | 改成一次流式同时收集 |
| 🔴 | **无对话记忆** | 每次关掉就忘了你是谁 | 加 ChromaDB 向量记忆 |
| 🔴 | **工具调用不稳定** | LLM 格式错就崩 | 加参数校验 + 自动重试 |
| 🟠 | **服务器断连无恢复** | 关了就没了 | 系统托盘 + 开机自启 |
| 🟠 | **STT 模型未预加载** | 第一次用等很久 | 启动时预热模型 |
| 🟠 | **无打断检测** | Javis 说话不能停 | 麦克风实时监听 |
| 🟡 | **摄像头无预览** | 看不到拍了什么 | 前端加实况窗 |
| 🟡 | **错误信息技术化** | 用户看不懂 | 转成自然语言 |
| 🟡 | **无多用户支持** | 只服务一个人 | 加用户身份识别 |
| ⚪ | **UI 单调** | 不像 JARVIS | 蓝橙动效/全息风 |
