# Javis 智能体核心架构 v3.0

## 设计哲学
Javis 是一个能自我学习、自我修正、跨平台的人工智能体。
他不是"工具集合"，而是一个**不断进化的数字生命**。

## 三层架构

```
┌─────────────────────────────────────────────┐
│  ③ 元认知层 (Meta-Cognition Layer)           │
│  自我反思 · 错误修正 · 策略进化              │
├─────────────────────────────────────────────┤
│  ② 知识层 (Knowledge Layer)                  │
│  短期记忆(对话) · 长期记忆(知识库) · 技能树  │
├─────────────────────────────────────────────┤
│  ① 执行层 (Execution Layer)                  │
│  run_code · 预置工具 · 跨平台抽象 · 权限管理 │
└─────────────────────────────────────────────┘
```

## 自学习机制

每一次对话 = 一次学习机会：

```
对话结束 → 提取"学到了什么" → 存储到知识库
执行失败 → 分析错误原因 → 修正策略 → 存入经验库
用户纠正 → 理解意图 → 更新行为模式
```
